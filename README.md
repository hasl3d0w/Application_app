   🌍My Portfolio🌍
This project is a Multipage Web Application built using the Streamlit framework in Python. It showcases a personal portfolio with multiple interactive pages, clean UI/UX design, and dynamic user features.

The application demonstrates creativity, technical skills, and the ability to design a user-friendly web interface.

🚀 Features
🏠 Home Page – Introduction and interactive survey
👤 About Page – Personal information, skills, and fun facts
💻 Projects Page – Sample projects including:
Calculator App (interactive)
📬 Contact Page – Contact form with input validation
🎯 Key Highlights
Multipage navigation using Streamlit
Clean and responsive layout
Interactive components (buttons, inputs, select boxes)
Organized project structure
